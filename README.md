# 广西文化和旅游网站

一个展示广西壮族自治区丰富文化和旅游资源的单页网站。

## 功能特点

### 🎨 页面模块
- **首页** - 展示广西旅游轮播图、走进广西介绍、山海佳肴美食卡片、特色区域展示
- **八桂风光** - 展示广西著名景点（漓江山水、德天瀑布、北海银滩等）
- **文化解码** - 展示广西民族风情、非遗瑰宝、建筑古韵、民俗节庆、方言曲艺、诗词艺文
- **风物档案** - 展示广西山水名物、探访实录、人间风物
- **旅游指南** - 包含新闻资讯和互动地图功能
- **在线咨询** - 留言表单功能

### 🤖 AI智能助手
- 页面滚动时显示悬浮AI助手
- 悬停展开完整内容
- 点击打开聊天窗口
- 支持智能问答（景点推荐、美食推荐、行程规划、交通住宿建议）

### 🗺️ 互动地图
- 4:3比例地图展示
- 支持景点、餐饮、酒店分类切换
- 地图标记点悬停显示信息
- 点击标记点显示详细信息弹窗

### ✨ 特效功能
- 首页滚动动画效果
- 美食卡片展开交互
- 图片悬停放大效果
- 响应式导航栏

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计（Flexbox、Grid、动画效果）
- **JavaScript** - 交互逻辑
- **Font Awesome** - 图标库

## 部署说明

### GitHub Pages 部署

1. **创建GitHub仓库**
   - 在GitHub上创建一个新仓库
   - 推荐使用 `username.github.io` 格式命名

2. **上传文件**
   - 将所有文件上传到仓库根目录
   - 确保 `index.html` 在根目录

3. **启用GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source选择 `main` 分支，目录选择 `/ (root)`
   - 点击 Save

4. **访问网站**
   - 部署完成后访问 `https://username.github.io`

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/username/username.github.io.git

# 进入目录
cd username.github.io

# 使用浏览器打开
start index.html
```
